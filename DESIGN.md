# Design System: TBN Project
**Project ID:** 

## 1. Visual Theme & Atmosphere
Clean, modern, and professional. High contrast with subtle gradients.

## 2. Color Palette & Roles
- **Primary** (#007bff) – Main brand color
- **Secondary** (#6c757d) – Supporting color
- **Background** (#f8f9fa) – Page background
- **Surface** (#ffffff) – Card and container background
- **Text Primary** (#212529) – Main text color

## 3. Typography Rules
System fonts (Inter, Roboto, sans-serif). Sizes: H1 (32px), Body (16px).

## 4. Component Stylings
* **Buttons:** Rounded corners (8px), solid primary color.
* **Cards:** Subtle box shadow (0 4px 6px rgba(0,0,0,0.1)), white background.
* **Inputs:** Bordered with focus state highlight.

## 5. Layout Principles
Mobile-responsive, centered containers, generous whitespace.
