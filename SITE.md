# Project Vision & Constitution

> **AGENT INSTRUCTION:** Read this file before every iteration. It serves as the project's "Long-Term Memory."

## 1. Core Identity
* **Project Name:** The Global Notariat (TBN)
* **Mission:** Providing timeless notarial services for a connected world through high-tech, secure, and accessible solutions.
* **Target Audience:** Remote and local clients needing legal document notarization, loan signings, and apostille services.
* **Voice:** Professional, Authoritative, Trustworthy, and Modern.

## 2. Visual Language
* **The "Vibe":** Award-winning, premium, and sleek.
* **Colors:** Navy (#1a2b4b) and Gold (#b2945e).
* **Typography:** Modern Sans-serif (Geist).
* **Animations:** Fluid and premium transitions using Framer Motion.

## 3. Architecture
* **Stack:** Next.js (App Router), Tailwind CSS, Prisma, MySQL, NextAuth, Nodemailer.
* **Database:** `tbn` (MySQL on 127.0.0.1).
* **SMTP:** Gmail (smartestdevelopers@gmail.com).

## 4. Live Sitemap
* [/] `Home` - Animated hero, about, services overview, and contact integrated.
* [/contact] `Contact Us` - Dedicated contact page with validation and email triggers.
* [/admin] `Dashboard` - Administrative overview and statistics.
* [/admin/messages] `Messages` - CRUD management for contact inquiries.
* [/admin/services] `Services Management` - Control panel for service listings.
